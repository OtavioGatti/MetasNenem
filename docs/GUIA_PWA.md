# 📱 MetasNenem PWA - Guia de Instalação

## O que é PWA?

**PWA (Progressive Web App)** transforma o MetasNenem em um aplicativo instalável no seu celular, funcionando até **offline**!

---

## 📲 Como Instalar

### Android (Chrome)

1. Acesse o MetasNenem no **Chrome**
2. Automaticamente aparecerá: **"Instalar MetasNenem?"**
3. Toque em **Instalar**
4. Pronto! O ícone 💕 aparecerá na sua tela inicial

### iPhone/iPad (Safari)

1. Acesse o MetasNenem no **Safari**
2. Toque no botão **Compartilhar** (⎋)
3. Role para baixo e toque em **"Adicionar à Tela de Início"**
4. Confirme o nome e toque em **"Adicionar"**
5. Pronto! O ícone aparecerá na sua tela inicial

### Desktop (Chrome/Edge)

1. Acesse o MetasNenem
2. Clique no ícone 📲 **"Instalar App"** no canto inferior direito
3. Ou clique no ícone de instalação na barra de endereço
4. Confirme a instalação

---

## ✨ Funcionalidades PWA

| Recurso | Descrição |
|---------|-----------|
| 📱 **Instalável** | Ícone na tela inicial como app nativo |
| 🌐 **Offline** | Funciona sem internet (dados salvos localmente) |
| ⚡ **Rápido** | Carrega instantaneamente |
| 🎨 **Tela Cheia** | Sem barra de navegador |
| 🔔 **Notificações** | Alertas quando parceiro completa tarefa |
| 🔄 **Auto-Update** | Atualiza automaticamente |

---

## 📴 Modo Offline

Quando estiver **offline**:
- ✅ Pode criar tarefas
- ✅ Pode completar tarefas  
- ✅ Dados salvos localmente
- 🔄 Sincroniza automaticamente quando voltar online

---

## 🎯 Atalhos (Android)

Mantenha pressionado o ícone do app para acessar:
- ✅ **Nova Tarefa** - Cria tarefa rápida
- 🎯 **Novo Desafio** - Cria desafio de casal

---

## 🐛 Troubleshooting

### App não aparece para instalar?
- Verifique se está acessando via **HTTPS**
- No iOS: use **Safari** (não Chrome)
- Tente limpar cache do navegador

### Não sincroniza?
- Verifique conexão com internet
- Aguarde alguns segundos
- Force refresh: puxe para baixo (mobile) ou F5 (desktop)

### Quer reinstalar?
- Remova o app da tela inicial
- Acesse o site novamente
- Siga os passos de instalação
